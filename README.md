## BashTeacher

Kurs: II1305 ICT Project

BashTeacher är en webbaserad plattform som hjälper användare att lära sig Bash-kommandon genom interaktiva övningar och spelifiering. Projektet utvecklades i grupp enligt agila arbetsmetoder.
